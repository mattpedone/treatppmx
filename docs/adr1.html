How to initialize cluster counts

##### Pseudocode

  • n N number of observations
  • i, ii {1, , n} indices
  • nh, Si Rn vectors

nh is the vector of clusters cardinality, while Si is the vector of labels (indicates for each subject to which clusters it belongs to)

for i in n
for ii in n
if Si(i) == ii+1
nh(ii) += 1

NB

Il vettore delle label Si indica i cluster non come C++ (potrebbe creare confusione), quindi le etichette inizieranno da 1. Per verificare se un cluster popolato o un singoletto user il vettore Si ed il suo indice come indice:

if(nh(Si(i)-1) > 1)
se questa condizione vera lelemento i-esimo appartiene ad un non-singleton
